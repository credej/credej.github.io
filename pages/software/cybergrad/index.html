cybergrad

The 'CyberGrad' software application was created to train users on the basics of cyber security through interactive lessons and problems. For the project our team considered using a pre-built LMS (Learning Management Software) but instead decided to create a new custom piece of software. This project was a challenge because of the rapid design & development pace - from inception to shipping the final CyberGrad product was completed in just under six months.

The training software allows teachers to assign lessons and quizzes and for students to view and work through those assignments. Both students and teachers also have account dashboards to manage course progression. My responsibilities for the CyberGrad project included:

My main responsibility in this project was to design new UI/UX features for the cyber security training application. Creating a custom application rather than starting with an out-of-the-box software solution allowed for a higher quality design of the user interface and ultimately a better experience for the user. In addition to assisting with the app's UI/UX, I was responsible for creating the logo and badge sets
