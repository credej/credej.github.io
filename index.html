Senitron: Inventory Management for Retail & Warehouses

A series of physical installations that utilize RFID tracking antennas to monitor either a retail or warehouse's inventory. I was responsible for both designing and developing multiple user interfaces for the inventory tracking software.

The RFID tag tracking system delivers a hands-free solution for tracking items. This allows staff to stay on top of inventory levels 24/7 with extremely high accuracy. After installing hundreds of RFID tracking antennas at one of Intuitive's San Jose warehouses, a back-end developer and myself implemented a custom software application to track inventory for their da Vinci surgical robot system. I designed and developed the user interface and experience. The 200,000 square feet warehouse enables asset tracking with excellent granularity down to particular shelves, aisles, and shelf-levels.

American Apparel stores traditionally used a manual item count system which takes roughly 240 man-hours per month to keep track of per store, leaving lots of potential for human error. After automating their inventory process using custom-built software to track RFID tags, employees are now able to monitor inventory statistics in real-time as well as locate specific items at any time.

For Verizon I helped develop a prototype dashboard for Verizon's many retail locations keep track of their in-store inventory in real time and 24/7. I was responsible for designing and developing the UI for the employees to search their store's inventory.

sports_hockeyAbout Me

Both designing & developing web application software have changed a lot in the last decade.  If you prefer, you don't even need a CSS file anymore! And it's going to start changing even more rapidly.  While certain things about humanity remain bleak, we are simultaneously experiencing an exponential growth in technology that has the potential to drastically improve quality of life for all of us.  Open source software is alive and well serving billions (billions *daily*, take that McDonald's) and humans have figured out how to decentralize finance and bank themselves.  The internet has always been a double-edged sword, but now, finally, communication cannot be easily stopped.  With a plethora of frameworks, design systems, and other tools to choose from, the designer and developer have never had it so good!

My next goal is to find an interesting project that ideally helps unite or empower people in some way.  My creativity and drive for continuous learning has allowed me to adapt quickly in new environments.  What I use for software development include HTML, CSS (SASS preferred but TailWind is also cool).  For this website I used TailwindCSS as it's great for rapid developemnt and I was able to update and improve the site quickly, as well as a bit of javascript for the dynamic elements and of course git for version control.  For UI/UX design I use either Figma or Adobe XD.  I also have experience with Google Adwords & Analystics, Wordpress, Mailchimp, Adobe Suite, and Microsoft Office.  I use a Macbook personally, but I'm comfortable developing software in all operating systems.

It has become clear that open source software is the path forward when it relates to many industries, especially in software development. Nearly 100% of web applications leverage at least one form of open source software. Personally I'm becoming more dependent on OS software everyday, but it's ok - because there is no corporation to start charging or rug pull me. As long as the software is considered useful, and there are contributors, the project will live. Beautiful.

Aside from annoying my daughter, my hobbies include practicing Brazilian jiu jitsu, playing video games, and watching contact sports like mixed martial arts and ice hockey.
