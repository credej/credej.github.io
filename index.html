Senitron RFID Inventory Tracking Software

User Interface and Experience Design

Senitron develops radio frequency identification (RFID) hardware and software to track inventory with over 99% accuracy in real time. They provide continuous monitoring of RFID tags attached to inventory items in dense populations (in some cases 100,000+ tags) using a network of tag-reading antennas and custom built software applications. I was responsible for translating client requirements into a one page web interface. Below are screenshots of the web applications developed for early adopting companies.  Visit site...

This web application was developed for a major U.S. wireless carrier's flagship store. Inventory is tracked continuously, with the application directing employees to specific action items as well as inventory metrics. (rollover image for more)
A robot-assisted surgery manufacturer manually manages their product shelf life using a first-in, first-out method of organization. With the Senitron system installed, every item in the warehouse is tagged upon arrival. Item location, time in, time out, and other statistics are recorded continuously in real time as inventory changes. The user interface above represents the floor layout of one of their inventory storage warehouses.

Covisus Quantitative Optical Analysis

Product Design & Development - User Interface and Experience Design

Material surfaces have a randomized structure at a microscopic scale. Like a fingerprint, these patterns are unique and intrinsic to each individual item and cannot be predicted or replicated. Covisus, through its DTEK technology, can track physical items to original manufacturing condition by using algorithms to compare parts against a golden sample and determine product quality conformance. My task was to design and develop the hardware and software for scanning and analyzing components.   Read more...

The DTEK optically analyzes the surface of an electronic component and outputs unambiguous quantitative information about the surface for the purpose of comparative analysis.
A web application was created to upload and analyze surface texture scans and then determine product conformance and authenticity by comparing to a golden part. A report is generated from the analyzed set of chips.
